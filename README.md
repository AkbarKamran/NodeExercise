
# Node Task




## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`ACCESS_TOKEN_KEY`




## Run Project

First:

```bash
  npm install
```

To Run Project in Development:
```bash
  npm run dev
```

Build:
```bash
  npm run Build
  npm start
```



## Swagger Endpoint
